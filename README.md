# RIZZK Calculator

Canonical repository for RIZZK Calculator, a production-focused risk and position-sizing tool for day traders.

## What it does

RIZZK helps traders calculate position size, risk dollars, and risk/reward outcomes using disciplined, repeatable math.

## Core features

- Position sizing from account size and risk constraints
- Long and short scenario support
- Profit target calculations
- CSV export for journaling and analysis
- Unit-tested calculation engine

## Stack

- Python
- Streamlit
- Pandas
- Pytest

## Quickstart

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tests

```bash
pytest test_risk_reward.py -v
```
